# InternTrack – Internship Tracking Web App

InternTrack is a full-stack web application that allows users to track and manage their internship applications.

## Features
- User registration and login
- Add, edit, delete internship applications
- Track status (Applied, Interview, Offer, Rejected)
- Dashboard with application list and totals

## Technologies
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## How to Run
1. Install XAMPP
2. Start Apache & MySQL
3. Place project in htdocs
4. Create database: interntrack
5. Run at http://localhost/InternTrack/
